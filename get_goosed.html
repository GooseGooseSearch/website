GooseGooseSearch

Get Goosed!

The GooseGooseSearch client is written in the Terra programming language. With that in mind, GooseGooseSearch should work out of box on any unix-based system. Currently, your biggest worry is if you're a Windows user: Your operating system isn't supported by GTK+3, the toolkit that we use to create and manage windows and widgets in them.

If you're having trouble getting a working Terra install, you can just download one of our binaries from the Releases page of the GitHub repository.

If you want to download GooseGooseSearch for your system, first install the Terra programming language, get yourself a copy of our git repo and run 'goose'.

For your privacy, we suggest that you configure your client to connect to your searx instance of choice over Tor.
