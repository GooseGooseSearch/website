GooseGooseSearch

Run your own server

GooseGooseSearch itself is not a search engine! We use the searx metasearch engine. If you want to set up your own server, just follow the directions on thier GitHub repo! Any searx instance is compatible with GooseGooseSearch.

Once your searx instance is up, just point your client to it using the Manual URL entry field in the client. It's that easy.
